# 📦 Stock Management Application

## 📝 Overview
This is a **desktop stock management application** developed during my summer internship at **PolySoft Informatique**.  
The project was built using **C# (.NET)**, **SQL Server**, and **DevExpress** with the **MVVM architecture**.  

## 🚀 Features
- 📊 Manage articles, clients, suppliers, and stock movements.  
- 📝 Create and manage purchase orders, delivery notes, and entry vouchers.  
- 🔍 Integrated search with DevExpress controls (SearchLookUpEdit, etc.).  
- 📑 Printable reports with DevExpress XtraReport.  
- ⚡ Real-time stock updates and validation.  

## 🛠️ Tech Stack
- **C# / .NET (WinForms + MVVM)**
- **SQL Server (Entity Framework Core)**
- **DevExpress UI Components**
- **Visual Studio 2022**
